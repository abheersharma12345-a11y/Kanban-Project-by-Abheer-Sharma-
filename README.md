# Kanban Board – Task Management Application

A simple **Kanban-style task manager** built to organize tasks efficiently using a visual workflow.

This project allows users to create tasks and move them between different stages such as **To-Do, In Progress, and Completed**.

---

## Project Overview

This application demonstrates a modern **single-page task management system** designed with an intuitive interface and smooth interactions.
Users can add tasks, track progress, and manage their workflow using a Kanban board layout.

---

## Features

* Create and manage tasks easily
* Move tasks between **To-Do, In Progress, and Completed**
* Clean and responsive user interface
* Smooth drag-and-drop task management
* Modern UI with animations and gradient design

---

## Technologies Used

* **React**
* **TypeScript**
* **Vite**
* **Tailwind CSS**
* **shadcn-ui**

---

## Installation & Setup

To run the project locally:

```bash
# Clone the repository
git clone <YOUR_GIT_URL>

# Go to the project folder
cd <PROJECT_FOLDER>

# Install dependencies
npm install

# Run the development server
npm run dev
```

The application will start on the local development server.
